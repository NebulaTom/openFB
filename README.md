## 2，轴控功能块

使用结构体数组方式实现，创建好相应的结构体数组变量，传给fb块即可，另外再将轴号传给fb块即可；

## 3，轴位置存取

使用指针方式，实现按照编号对寄存器（D地址）内的位置数据进行取出或写入（获取位置）；

## 4， 气缸控制

以结构体数组的方式实现，创建好相应的电磁阀结构体数组变量后，传给fb块，另外将输入输出点也传给fb块即可

# 注意

想要查看功能块全部的输入输出，可以再创建的结构体变量中查看，

结构体使用时使用  变量名+“.” 引出，如： set 气缸[0].i_控制输入 ，就是让气缸动作，轴控同理
